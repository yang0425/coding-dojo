### Part 1
Senerio 1: if it is divisible by three it is "Fizz".
```
Given: number 3
When: transfer
Then: Fizz
```
Senerio 2: if it is divisible by five it is "Buzz".
```
Given: number 5
When: transfer
Then: Buzz
```
Senerio 3: if it is both "Fizz" and "Buzz" it is "FizzBuzz".
```
Given: number 15
When: transfer
Then: FizzBuzz
```
