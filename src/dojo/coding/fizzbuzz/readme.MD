### Part 1
Senerio 1: if it is divisible by three it is "Fizz".
```
Given: number 3
When: is Fizz?
Then: true

Given: number 2
When: is Fizz?
Then: false
```
Senerio 2: if it is divisible by five it is "Buzz".
```
Given: number 5
When: is Buzz?
Then: true

Given: number 2
When: is Buzz?
Then: false
```
Senerio 3: if it is both "Fizz" and "Buzz" it is "FizzBuzz".
```
Given: number 15
When: is FizzBuzz?
Then: true

Given: number 5
When: is FizzBuzz?
Then: false
```
